1. рЃАрЃарЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў (UI/UX рЃљрЃдрЃгрЃћрЃарЃљ)
рЃцрЃћрЃарЃћрЃЉрЃў рЃБрЃцрЃарЃЮ рЃџрЃљрЃЏрЃљрЃќрЃљрЃЊ рЃњрЃљрЃЋрЃ«рЃљрЃЊрЃћ (рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћ рЃЌрЃЉрЃўрЃџрЃўрЃАрЃўрЃА рЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃърЃљрЃџрЃўрЃбрЃарЃљ: рЃЏрЃгрЃЋрЃљрЃюрЃћ рЃърЃљрЃарЃЎрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃџрЃБрЃарЃ»рЃў рЃЏрЃбрЃЎрЃЋрЃарЃўрЃАрЃЌрЃЋрЃўрЃА, рЃюрЃљрЃарЃўрЃюрЃ»рЃўрЃАрЃцрЃћрЃарЃў рЃћрЃюрЃћрЃарЃњрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА). рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљ рЃцрЃћрЃарЃерЃў (рЃЏрЃљрЃњ. рЃюрЃљрЃарЃўрЃюрЃ»рЃўрЃАрЃцрЃћрЃарЃў), рЃЉрЃљрЃюрЃћрЃарЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљ рЃцрЃћрЃарЃерЃў (рЃЏрЃБрЃЦрЃў рЃџрЃБрЃарЃ»рЃў gradient). рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃўрЃюрЃЮрЃЋрЃљрЃфрЃўрЃћрЃЉрЃў: AI-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў (рЃЏрЃљрЃњ. "рЃерЃћрЃюрЃЌрЃЋрЃўрЃА рЃерЃћрЃарЃЕрЃћрЃБрЃџрЃў рЃбрЃБрЃарЃћрЃЉрЃў" OpenAI-рЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃўрЃЌ), AR preview (рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃќрЃћ рЃЎрЃљрЃЏрЃћрЃарЃўрЃЌ рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃ«рЃћрЃЊрЃў, рЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА). рЃБрЃЎрЃљрЃюрЃљ рЃЊрЃљрЃцрЃљ (рЃЉрЃћрЃЦрЃњрЃарЃљрЃБрЃюрЃЊрЃў) рЃбрЃБрЃарЃўрЃќрЃЏрЃќрЃћ рЃЏрЃўрЃЏрЃљрЃюрЃўрЃерЃюрЃћрЃЉрЃћрЃџрЃў: subtle рЃЌрЃЉрЃўрЃџрЃўрЃАрЃўрЃА рЃ«рЃћрЃЊрЃћрЃЉрЃў (рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃюрЃљрЃарЃўрЃДрЃљрЃџрЃљ рЃљрЃю рЃФрЃЋрЃћрЃџрЃў рЃЌрЃЉрЃўрЃџрЃўрЃАрЃў), рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф fade-рЃўрЃЌ рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃљ рЃЏрЃФрЃўрЃЏрЃћ (рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ low-opacity рЃўрЃЏрЃўрЃ»рЃћрЃЉрЃў CSS-рЃўрЃЌ). рЃБрЃАрЃћрЃарЃўрЃА рЃЏрЃўрЃћрЃа рЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў рЃцрЃЮрЃбрЃЮ/рЃЋрЃўрЃЊрЃћрЃЮ: рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ upload рЃдрЃўрЃџрЃљрЃЎрЃў (Supabase Storage-рЃўрЃЌ), рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃў (textarea), рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ (1-5 рЃЋрЃљрЃарЃАрЃЎрЃЋрЃџрЃљрЃЋрЃў + emoji reactions). рЃАрЃљрЃЕрЃБрЃЦрЃљрЃарЃў: rewards рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ (badges рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ "рЃбрЃЮрЃъ рЃЎрЃЮрЃюрЃбрЃарЃўрЃЉрЃБрЃбрЃЮрЃарЃў" рЃљрЃЦрЃбрЃўрЃБрЃарЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃцрЃљрЃАрЃўрЃљрЃюрЃў рЃЋрЃћрЃарЃАрЃўрЃљ рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃерЃў РђЊ unlock premium badges рЃљрЃю ad-free). рЃћрЃюрЃљ: рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ рЃЏрЃарЃљрЃЋрЃљрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў РђЊ рЃЦрЃљрЃарЃЌрЃБрЃџрЃў (рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў), рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў, рЃарЃБрЃАрЃБрЃџрЃў (рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ i18next React-рЃерЃў, Supabase-рЃерЃў рЃерЃћрЃўрЃюрЃљрЃ«рЃћ рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃћрЃЉрЃў JSON-рЃерЃў). рЃљрЃърЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃдрЃЏрЃЮрЃљрЃЕрЃћрЃюрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃћрЃюрЃљрЃА (browser locale) рЃЊрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃЏрЃўрЃАрЃфрЃћрЃЏрЃА рЃерЃћрЃфрЃЋрЃџрЃљрЃА рЃърЃарЃЮрЃцрЃўрЃџрЃерЃў.
рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃџрЃћрЃўрЃљрЃБрЃЌрЃў рЃЊрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў:

рЃ░рЃћрЃЊрЃћрЃарЃў: рЃџрЃЮрЃњрЃЮ рЃЏрЃљрЃарЃфрЃ«рЃюрЃўрЃЋ (рЃЌрЃЉрЃўрЃџрЃўрЃАрЃўрЃА рЃАрЃўрЃџрЃБрЃћрЃбрЃў + рЃбрЃћрЃЦрЃАрЃбрЃў), рЃФрЃћрЃЉрЃюрЃўрЃА рЃЉрЃљрЃарЃў рЃфрЃћрЃюрЃбрЃарЃерЃў (рЃљрЃЋрЃбрЃЮрЃЎрЃЮрЃЏрЃърЃџрЃўрЃбрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃЌ), рЃЏрЃћрЃюрЃўрЃБ рЃЏрЃљрЃарЃ»рЃЋрЃюрЃўрЃЋ: рЃарЃБрЃЎрЃљ (рЃџрЃБрЃарЃ»рЃў рЃўрЃЎрЃЮрЃюрЃљ), рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў (рЃюрЃљрЃарЃўрЃюрЃ»рЃўрЃАрЃцрЃћрЃарЃў), рЃърЃарЃЮрЃцрЃўрЃџрЃў (рЃЏрЃгрЃЋрЃљрЃюрЃћ), рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў, рЃћрЃюрЃўрЃА рЃАрЃЋрЃўрЃЕрЃћрЃарЃў (KA/EN/RU dropdown). рЃЉрЃћрЃЦрЃњрЃарЃљрЃБрЃюрЃЊрЃў: light gradient рЃЏрЃгрЃЋрЃљрЃюрЃћ-рЃџрЃБрЃарЃ»рЃў рЃбрЃБрЃарЃўрЃАрЃбрЃБрЃџрЃў рЃўрЃЏрЃўрЃ»рЃћрЃЉрЃўрЃЌ (рЃЏрЃљрЃњ. blurred рЃцрЃЮрЃбрЃЮ рЃФрЃЋрЃћрЃџрЃў рЃЌрЃЉрЃўрЃџрЃўрЃАрЃўрЃА).
рЃцрЃБрЃбрЃћрЃарЃў: рЃўрЃЎрЃЮрЃюрЃћрЃЉрЃў рЃЦрЃЋрЃћрЃЊрЃљ рЃЉрЃљрЃарЃерЃў рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃќрЃћ, рЃърЃџрЃБрЃА рЃћрЃюрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃў.
рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃћрЃЎрЃарЃљрЃюрЃў:
рЃбрЃЮрЃъ: Carousel рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃЌ (рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃў рЃБрЃАрЃћрЃарЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃћрЃЉрЃўрЃЌ, рЃЌрЃБ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА).
рЃерЃБрЃљ: Feed рЃЎрЃљрЃарЃЊрЃћрЃЉрЃўрЃЌ (рЃљрЃ«рЃљрЃџрЃў рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў/рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў), рЃЌрЃўрЃЌрЃЮ рЃЎрЃљрЃарЃЊрЃў: рЃцрЃЮрЃбрЃЮ/рЃЋрЃўрЃЊрЃћрЃЮ (рЃБрЃАрЃћрЃарЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў), рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ, рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ (рЃЋрЃљрЃарЃАрЃЎрЃЋрЃџрЃљрЃЋрЃћрЃЉрЃў), рЃџрЃљрЃўрЃЦрЃў. рЃўрЃюрЃЮрЃЋрЃљрЃфрЃўрЃљ: AI рЃдрЃўрЃџрЃљрЃЎрЃў "рЃерЃћрЃЏрЃЮрЃњрЃЌрЃљрЃЋрЃљрЃќрЃћ рЃбрЃБрЃарЃў" (рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃБрЃа рЃЏрЃљрЃарЃерЃарЃБрЃбрЃА).
рЃЉрЃћрЃЦрЃњрЃарЃљрЃБрЃюрЃЊрЃў: subtle рЃбрЃБрЃарЃўрЃАрЃбрЃБрЃџрЃў рЃўрЃЏрЃўрЃ»рЃў (рЃЏрЃљрЃњ. рЃЏрЃЌрЃљрЃгрЃЏрЃўрЃюрЃЊрЃўрЃА рЃ«рЃћрЃЊрЃў, opacity 0.1).

рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў: рЃЊрЃўрЃЊрЃў рЃцрЃЮрЃбрЃЮ/рЃЋрЃўрЃЊрЃћрЃЮ рЃърЃџрЃћрЃўрЃћрЃарЃў рЃќрЃћрЃЏрЃЮрЃЌ (рЃБрЃАрЃћрЃарЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў рЃЏрЃћрЃЊрЃўрЃљ рЃњрЃљрЃџрЃћрЃарЃћрЃљрЃерЃў), рЃљрЃдрЃгрЃћрЃарЃљ (рЃЏрЃарЃљрЃЋрЃљрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў), рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ (рЃарЃћрЃўрЃбрЃўрЃюрЃњрЃў + рЃцрЃЮрЃарЃЏрЃљ рЃерЃћрЃюрЃў рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА), рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃўрЃА рЃАрЃћрЃЦрЃфрЃўрЃљ (рЃџрЃўрЃАрЃбрЃў + add comment), рЃарЃБрЃЎрЃљ. рЃўрЃюрЃЮрЃЋрЃљрЃфрЃўрЃљ: AR button (рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃќрЃћ рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃбрЃБрЃарЃА, Unity рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃўрЃЌ рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃерЃў). рЃАрЃљрЃЕрЃБрЃЦрЃљрЃарЃў: рЃЌрЃБ рЃЊрЃљрЃърЃЮрЃАрЃбрЃљрЃЋ рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃА, рЃЏрЃўрЃўрЃдрЃћрЃЉ badge (рЃЏрЃљрЃњ. рЃ░рЃћрЃЊрЃћрЃарЃў рЃљрЃЊрЃЏрЃўрЃюрЃерЃў: рЃўрЃњрЃўрЃЋрЃћ рЃарЃЮрЃњрЃЮрЃарЃф рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ "Admin Mode" рЃџрЃћрЃўрЃЉрЃџрЃў рЃЊрЃљ logout.
рЃЉрЃћрЃЦрЃњрЃарЃљрЃБрЃюрЃЊрЃў: рЃЏрЃАрЃБрЃЉрЃБрЃЦрЃў gradient рЃџрЃБрЃарЃ»рЃў-рЃЏрЃгрЃЋрЃљрЃюрЃћ, рЃЌрЃЉрЃўрЃџрЃўрЃАрЃўрЃА рЃбрЃБрЃарЃўрЃАрЃбрЃБрЃџрЃў рЃўрЃЏрЃўрЃ»рЃћрЃЉрЃўрЃЌ (opacity 0.2), рЃарЃљрЃЌрЃљ рЃљрЃа рЃерЃћрЃљрЃцрЃћрЃарЃ«рЃЮрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ.

рЃцрЃћрЃарЃћрЃЉрЃў рЃЊрЃљ рЃАрЃбрЃўрЃџрЃў: рЃўрЃњрЃўрЃЋрЃћ рЃърЃљрЃџрЃўрЃбрЃарЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃЊрЃЏрЃўрЃюрЃерЃў рЃБрЃцрЃарЃЮ "рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃБрЃџрЃў" РђЊ рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў рЃгрЃўрЃЌрЃћрЃџрЃў (рЃЊрЃћрЃџрЃћрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА), рЃЏрЃгрЃЋрЃљрЃюрЃћ (рЃАрЃћрЃўрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА), рЃЉрЃљрЃюрЃћрЃарЃћрЃЉрЃў рЃЏрЃБрЃЦрЃў рЃџрЃБрЃарЃ»рЃў. рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў (fade-in рЃбрЃљрЃЉрЃџрЃћрЃЉрЃќрЃћ).

рЃљрЃЊрЃЏрЃўрЃюрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў (рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃљрЃЊ РђЊ рЃАрЃарЃБрЃџрЃў рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃќрЃћ):
рЃљрЃЊрЃЏрЃўрЃюрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ, рЃарЃљрЃф рЃљрЃърЃА рЃљрЃЦрЃЋрЃА (рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў, рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў, рЃБрЃАрЃћрЃарЃћрЃЉрЃў, рЃарЃћрЃЎрЃџрЃљрЃЏрЃљ, рЃЏрЃћрЃАрЃўрЃ»рЃћрЃЉрЃў рЃЊрЃљ рЃљ.рЃе.). рЃДрЃЋрЃћрЃџрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃбрЃљрЃЉрЃџрЃћрЃбрЃћрЃЉрЃўрЃЌ/рЃцрЃЮрЃарЃЏрЃћрЃЉрЃўрЃЌ, Supabase queries-рЃўрЃЌ (рЃЏрЃљрЃњ. select/update/delete). рЃЊрЃљрЃЋрЃДрЃљрЃЋрЃў рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃљрЃЊ:

Dashboard (рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃърЃљрЃюрЃћрЃџрЃў):
Overview: рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ (рЃбрЃЮрЃбрЃљрЃџ рЃБрЃАрЃћрЃарЃћрЃЉрЃў, рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў, рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў, рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃарЃћрЃЎрЃџрЃљрЃЏрЃћрЃЉрЃў). рЃњрЃарЃљрЃцрЃўрЃЎрЃћрЃЉрЃў (chart.js-рЃўрЃЌ: рЃБрЃАрЃћрЃарЃћрЃЉрЃўрЃА рЃќрЃарЃЊрЃљ, рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў).
Quick Actions: рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў рЃАрЃгрЃарЃљрЃцрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃЏрЃљрЃњ. "рЃљрЃ«рЃљрЃџрЃў рЃарЃћрЃЎрЃџрЃљрЃЏрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ", "рЃБрЃАрЃћрЃарЃўрЃА рЃЉрЃџрЃЮрЃЎрЃў").

Users (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ):
рЃБрЃАрЃћрЃарЃћрЃЉрЃўрЃА рЃџрЃўрЃАрЃбрЃў: рЃбрЃљрЃЉрЃџрЃљ (email, рЃАрЃљрЃ«рЃћрЃџрЃў, role, рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў, рЃљрЃЦрЃбрЃўрЃЋрЃЮрЃЉрЃљ). рЃФрЃћрЃЉрЃюрЃљ email-рЃўрЃЌ рЃљрЃю рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ.
рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў: рЃерЃћрЃфрЃЋрЃџрЃљ role-рЃўрЃА (рЃЏрЃљрЃњ. рЃњрЃљрЃ«рЃљрЃЊрЃћ moderator), рЃЉрЃџрЃЮрЃЎрЃў/рЃљрЃюрЃЉрЃџрЃЮрЃЎрЃў, рЃгрЃљрЃерЃџрЃљ. рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ (рЃЉрЃўрЃЮ, рЃљрЃЋрЃљрЃбрЃљрЃарЃў).
рЃЏрЃћрЃњрЃЮрЃЉрЃарЃћрЃЉрЃў/рЃЏрЃћрЃАрЃўрЃ»рЃћрЃЉрЃў: рЃБрЃАрЃћрЃарЃўрЃА рЃЏрЃћрЃњрЃЮрЃЉрЃарЃћрЃЉрЃўрЃА рЃџрЃўрЃАрЃбрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ/рЃгрЃљрЃерЃџрЃљ, рЃЏрЃћрЃАрЃўрЃ»рЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў (рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ/рЃгрЃљрЃерЃџрЃљ рЃАрЃърЃљрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА).

Content (рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃарЃљрЃфрЃўрЃљ):
рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА/рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃџрЃўрЃАрЃбрЃў: рЃбрЃљрЃЉрЃџрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃћрЃюрЃбрЃарЃўрЃЌ (рЃАрЃљрЃ«рЃћрЃџрЃў, рЃљрЃдрЃгрЃћрЃарЃљ, рЃБрЃАрЃћрЃарЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў рЃцрЃЮрЃбрЃЮ/рЃЋрЃўрЃЊрЃћрЃЮ, рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃў, рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃћрЃЉрЃў).
рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў: рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ (рЃерЃћрЃфрЃЋрЃџрЃљ рЃљрЃдрЃгрЃћрЃарЃўрЃА, рЃцрЃЮрЃбрЃЮрЃА), рЃгрЃљрЃерЃџрЃљ (рЃАрЃърЃљрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА), approve/deny рЃљрЃ«рЃљрЃџрЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃћрЃЉрЃў. рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃўрЃА/рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃарЃљрЃфрЃўрЃљ (рЃгрЃљрЃерЃџрЃљ). рЃАрЃљрЃЕрЃБрЃЦрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ рЃБрЃАрЃћрЃарЃћрЃЉрЃА (рЃЏрЃљрЃњ. badge-рЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃ«рЃћрЃџрЃўрЃЌ).

Events (рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ):
рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃџрЃўрЃАрЃбрЃў: рЃбрЃљрЃЉрЃџрЃљ (рЃЌрЃљрЃарЃўрЃдрЃў, рЃџрЃЮрЃЎрЃљрЃфрЃўрЃљ, рЃЏрЃЮрЃюрЃљрЃгрЃўрЃџрЃћрЃћрЃЉрЃў).
рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў: рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ/рЃерЃћрЃфрЃЋрЃџрЃљ/рЃгрЃљрЃерЃџрЃљ, RSVP-рЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ, рЃърЃарЃЮрЃЏрЃЮрЃБрЃерЃћрЃюрЃў (featured рЃњрЃљрЃ«рЃЊрЃћрЃА).

Ads (рЃарЃћрЃЎрЃџрЃљрЃЏрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ) РђЊ рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃљрЃЊ рЃерЃћрЃюрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃЌ рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў:
рЃарЃћрЃЎрЃџрЃљрЃЏрЃћрЃЉрЃўрЃА рЃџрЃўрЃАрЃбрЃў: рЃбрЃљрЃЉрЃџрЃљ (рЃбрЃўрЃърЃў: banner/sponsored, рЃЉрЃўрЃќрЃюрЃћрЃАрЃў, рЃЊрЃљрЃгрЃДрЃћрЃЉрЃўрЃА/рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў, рЃАрЃбрЃљрЃбрЃБрЃАрЃў: active/expired).
рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў: рЃљрЃ«рЃљрЃџрЃў рЃарЃћрЃЎрЃџрЃљрЃЏрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ (рЃцрЃЮрЃарЃЏрЃљ: рЃўрЃЏрЃўрЃ»рЃў/рЃбрЃћрЃЦрЃАрЃбрЃў, рЃърЃЮрЃќрЃўрЃфрЃўрЃљ РђЊ рЃ░рЃЮрЃЏрЃќрЃћ/рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ, рЃцрЃљрЃАрЃў рЃЌрЃБ рЃцрЃљрЃАрЃўрЃљрЃюрЃўрЃљ). рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ/рЃгрЃљрЃерЃџрЃљ, рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљ (рЃЏрЃљрЃњ. assign to specific page). рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ (clicks/views).
рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ: placeholder div-рЃћрЃЉрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃерЃў, Supabase table 'ads' (columns: id, content, position, start_date, end_date).

Analytics (рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ):
рЃарЃћрЃърЃЮрЃарЃбрЃћрЃЉрЃў: рЃБрЃАрЃћрЃарЃћрЃЉрЃўрЃА рЃљрЃЦрЃбрЃўрЃЋрЃЮрЃЉрЃљ, рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў/рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў, рЃарЃћрЃЎрЃџрЃљрЃЏрЃўрЃА рЃърЃћрЃарЃцрЃЮрЃарЃЏрЃљрЃюрЃАрЃў (рЃњрЃарЃљрЃцрЃўрЃЎрЃћрЃЉрЃў).

1. рЃЏрЃћрЃњрЃЮрЃЉрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ (Friends System)
Supabase Tables (рЃАрЃЦрЃћрЃЏрЃљ)
SQL-- рЃЏрЃћрЃњрЃЮрЃЉрЃарЃЮрЃЉрЃўрЃА junction table (рЃЏрЃарЃљрЃЋрЃљрЃџ-рЃЏрЃарЃљрЃЋрЃљрЃџ рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃљ)
create table user_friends (
  id bigint generated by default as identity primary key,
  user_id uuid references profiles(id) on delete cascade not null,
  friend_id uuid references profiles(id) on delete cascade not null,
  status text default 'pending' check (status in ('pending', 'accepted', 'rejected')), -- pending = рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃџрЃў
  requested_by uuid references profiles(id),
  created_at timestamptz default now(),
  unique(user_id, friend_id)
);

-- RLS рЃърЃЮрЃџрЃўрЃАрЃћрЃЉрЃў (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЏрЃћрЃњрЃЮрЃЉрЃарЃћрЃЉрЃў рЃЕрЃљрЃюрЃА)
alter table user_friends enable row level security;
create policy "Users can view their own friends" on user_friends for select
  using (auth.uid() = user_id or auth.uid() = friend_id);
create policy "Users can send friend requests" on user_friends for insert
  with check (auth.uid() = user_id);
create policy "Users can update/accept their requests" on user_friends for update
  using (auth.uid() = friend_id and status = 'pending');
